Your task is to calculate `ab` mod `1337` where `a` is a positive integer and

**Example 1:**

> Input: a = 2, b = [3]
>

**Example 2:**

> Input: a = 2, b = [1,0]
>
> Output: 1024

**Example 3:**

> Input: a = 1, b = [4,3,3,8,5,2]
>
> Output: 1

**Constraints:**

- `1 <= a <= 2^31 - 1`
- `1 <= b.length <= 2000`
- `0 <= b[i] <= 9`
- `b` does not contain leading zeros.
