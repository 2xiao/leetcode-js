                      # 2.7 堆

                      堆是一种特殊的树，只要满足以下两点的树，就是一个堆：

                      • 堆是一个完全二叉树；
                      • 堆中每一个节点的值都必须大于等于（或小于等于）其子树中每个节点的值。
