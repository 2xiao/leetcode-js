> Input: args = []
>
> Output: "Hello World"
>
> Explanation:
>

>
> f(); // "Hello World"

**Example 2:**

> Input: args = [{},null,42]
>
> Output: "Hello World"
>
> Explanation:
>
> const f = createHelloWorld();
>
> f({}, null, 42); // "Hello World"
>
> Any arguments could be passed to the function but it should still always return "Hello World".

**Constraints:**

- `0 <= args.length <= 10`
