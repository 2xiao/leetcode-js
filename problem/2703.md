to it.

**Example 1:**

> Input: args = [5]
>
> Output: 1
>
> Explanation:
>
> argumentsLength(5); // 1
>
> One value was passed to the function so it should return 1.

> Input: args = [{}, null, "3"]
>
> Output: 3
>
> Explanation:
>
> argumentsLength({}, null, "3"); // 3
>
> Three values were passed to the function so it should return 3.

**Constraints:**

- `args` is a valid JSON array
- `0 <= args.length <= 100`
