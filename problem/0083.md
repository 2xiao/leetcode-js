Given the `head` of a sorted linked list, _delete all duplicates such that
each element appears only once_. Return _the linked list **sorted** as well_.

**Example 1:**

![](https://assets.leetcode.com/uploads/2021/01/04/list1.jpg)

> Input: head = [1,1,2]
>
> Output: [1,2]

**Example 2:**

![](https://assets.leetcode.com/uploads/2021/01/04/list2.jpg)

> Input: head = [1,1,2,3,3]
>
> Output: [1,2,3]

**Constraints:**

- The number of nodes in the list is in the range `[0, 300]`.
- `-100 <= Node.val <= 100`
- The list is guaranteed to be **sorted** in ascending order.

```javascript
/**
 * @param {ListNode} head
 * @return {ListNode}
 */
var deleteDuplicates = function (head) {
	let res = new ListNode(0, head);
	let prev = res;
	while (prev.next && prev.next.next) {
		if (prev.next.val === prev.next.next.val) {
			// åˆ é™¤å’Œ prev.next é‡å¤çš„èŠ‚ç‚¹
			while (
				prev.next &&
				prev.next.next &&
				prev.next.val === prev.next.next.val
			) {
				prev.next = prev.next.next;
			}
		} else {
			prev = prev.next;
		}
	}
	return res.next;
};
```
