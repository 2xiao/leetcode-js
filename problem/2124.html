                                                                    Example 1:

                                                                    Explanation:

                                                                    Input: s = "abab"

                                                                    Output: false

                                                                    Explanation:

                                                                    There is an 'a' at index 2 and a 'b' at index 1.

                                                                    Hence, not every 'a' appears before every 'b' and we return false.

                                                                    Input: s = "bbb"
