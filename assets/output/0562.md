Given an `m x n` binary matrix `mat`, return _the length of the longest line
of consecutive one in the matrix_.

The line could be horizontal, vertical, diagonal, or anti-diagonal.

> Output: 3

>

**Constraints:**

  * `m == mat.length`
  * `n == mat[i].length`
  * `1 <= m, n <= 10^4`
  * `1 <= m * n <= 10^4`
  * `mat[i][j]` is either `0` or `1`.
