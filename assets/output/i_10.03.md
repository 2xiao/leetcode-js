times, write code to find an element in the array. You may assume that the

> 
> 
> 
> 
>

> 
> 
> 
> 
> 
> **Input** : arr = [15, 16, 19, 20, 25, 1, 3, 4, 5, 7, 10, 14], target = 11

> 
>
