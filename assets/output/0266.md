## 题目

Given a string `s`, return `true` _if a permutation of the string could form
a_ _**palindrome**_ _and_`false` _otherwise_.



**Example 1:**

> Input: s = "code"
> 
> Output: false

**Example 2:**

> Input: s = "aab"
> 
> Output: true

**Example 3:**

> Input: s = "carerac"
> 
> Output: true

**Constraints:**

  * `1 <= s.length <= 5000`
  * `s` consists of only lowercase English letters.
