**fibonacci sequence**.

**Example 1:**

> Input: callCount = 5
> 
> Output: [0,1,1,2,3]
> 
> Explanation:
>

>

> 
> gen.next().value; // 1
> 
> gen.next().value; // 2
> 
> gen.next().value; // 3

**Example 2:**

> Input: callCount = 0
> 
> Output: []
> 
> Explanation: gen.next() is never called so nothing is outputted

**Constraints:**

  * `0 <= callCount <= 50`
