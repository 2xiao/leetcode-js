# 7. 整数反转

🟠 <font color=#ffb800>Medium</font>&emsp; 🔖&ensp; [`数学`](/tag/math.md)&emsp; 🔗&ensp;[`力扣`](https://leetcode.cn/problems/reverse-integer) [`LeetCode`](https://leetcode.com/problems/reverse-integer)

## 题目

Given a signed 32-bit integer `x`, return `x` _with its digits reversed_. If
reversing `x` causes the value to go outside the signed 32-bit integer range
`[-231, 231 - 1]`, then return `0`.

**Assume the environment does not allow you to store 64-bit integers (signed
or unsigned).**



**Example 1:**

> Input: x = 123
> 
> Output: 321

**Example 2:**

> Input: x = -123
> 
> Output: -321

**Example 3:**

> Input: x = 120
> 
> Output: 21

**Constraints:**

  * `-231 <= x <= 231 - 1`
