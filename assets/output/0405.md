representation_. For negative integers, [two's
complement](https://en.wikipedia.org/wiki/Two%27s_complement) method is used.

should not be any leading zeros in the answer except for the zero itself.

solve this problem.



**Example 1:**

> Input: num = 26
> 
> Output: "1a"

**Example 2:**

> Input: num = -1
> 
> Output: "ffffffff"

**Constraints:**

  * `-231 <= num <= 231 - 1`
