**Note** that the factors should be in the range `[2, n - 1]`.

> Input: n = 1
> 
> Output: []

**Example 2:**

> Input: n = 12
> 
> Output: [[2,6],[3,4],[2,2,3]]

> Input: n = 37
> 
> Output: []

**Constraints:**

  * `1 <= n <= 10^7`
