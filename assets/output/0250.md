Given the `root` of a binary tree, return _the number of**uni-value**
__subtrees_.

> 
> Output: 4

**Example 2:**

> Input: root = []
> 
> Output: 0

**Example 3:**

> Input: root = [5,5,5,5,5,null,5]
> 
> Output: 6

**Constraints:**

  * `-1000 <= Node.val <= 1000`
